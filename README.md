## 問題意識と開発理由

大学院への進学・就職活動などの活動もあってか、私自身2022年は多くの出会いを経験した年でした。そしてそのほとんどが、オンライン上のものであり、ランダムに振り分けられた5人程度でのグループ活動でした。
おそらくどのようなグループの組み合わせにおいても、全員がきちんと平等に発言することは不可能であり、誰かがリーダーシップを取り始め、誰かが会話に入れないこともあるでしょう。
そのような役割分担は決して悪いことではありませんが、全員がより対等な立場でより平等に意見を出すことができる環境作りができないか、私は考えました。
そのような環境をつくるためには、お互いを知ることが大切であると考え、開発したのが**みんなのクイズ**です。

みんなのクイズは、Zoom などのWeb会議ツールと同時に使い、会話のきっかけを生み出し、話しやすい環境つくりの手助けをするものです。
コミュニケーションは大きく会話ベースとテキストベースに分けられますが、それぞれに良さと悪さがあり、人それぞれに得意不得意があります。
会話は、同時に複数人が行うことができず、単位時間あたりのコミュニケーション量が減少してしまいます。一方、その人の感情・人柄をより深く伝えることができます。
テキストでは、人数が増えれば触れるほど相対的に効率の良いコミュニケーションが実現できますが、感情が伝わりにくい側面があります。
そこで、Web会議ツールとみんなのクイズを同時に使うことで、お互いのコミュニケーション方法の欠点を補完し合うことができます。

## 遊び方
それでは遊び方を説明します。みんなのクイズは、複数人のユーザーで遊ぶことができますが、
ここでは2人のユーザーが参加する場合を想定し、２つのブラウザを使って双方の画面遷移を説明します。

  
<img width="900" alt="入室画面1" src="https://user-images.githubusercontent.com/104077941/201461894-c73cf6f6-d81c-4e2f-b109-9ff83a6ba693.png">
1. 「みんなのクイズ」のURLにアクセスすると、ログイン画面が表示されます。
<br><br><br>
  
<img width="900" alt="入室画面2" src="https://user-images.githubusercontent.com/104077941/201461928-82e34cd4-f1ed-439c-b2fc-4c00f6370c1e.png">
2. お好きな名前でログインしてください。ただし、他のユーザーと重複する名前を使用した場合は警告がでるので、再度異なる名前でログインしてください。

<img width="900" alt="入室ユーザー確認画面" src="https://user-images.githubusercontent.com/104077941/201461916-7ab6cceb-acac-42f2-b861-5fd2c4dfe02c.png">
3. 右上のボタンから、入室しているユーザーの一覧を表示できます。
<br><br><br>
  
  
<img width="900" alt="メッセージ送信画面" src="https://user-images.githubusercontent.com/104077941/201462052-4011f900-de77-4a94-8948-62dd96145c6d.png">
4. ルームの中では、入室しているユーザーと自由にメッセージのやりとりができます。
<br><br><br>
  
<img width="900" alt="出題画面1" src="https://user-images.githubusercontent.com/104077941/201462142-1d875e82-5283-4382-ab16-429c66ddc637.png">
5. このアプリのメイン機能であるクイズの出題、回答はコマンドを通して行うことができます。このルーム内で使用できるコマンドの一覧を、右上の「コマンド一覧」ボタンから確認できます。
<br>

* /answer : クイズに回答します
* /question : クイズを出題します
* /question-end : クイズの出題を締め切ります
<br>
この例では、左側のユーザーでクイズを出題し、右側のユーザーで回答することにします。
左側のユーザーで /question コマンドを使い、クイズを出題します。

## その他の機能
* 整合性のないコマンドの入力を受け付けない
  * 問題が出題されていないとき、/question-end が入力できない
  * 問題が出題されていないとき、/answer が入力できない
  * 問題が出題されているとき、/question が入力できない
  * 問題が出題されているとき、出題者から /answer が入力できない
  * 問題が出題されているとき、出題者以外から /question-end が入力できない
* ボタンを押さずとも、エンターキーで投稿できる
* 途中入室したユーザーでも、出題されているクイズを見ることができる
* メッセージには投稿時間が表示されるが、本日以外のメッセージには日付も表示される
* メッセージの改行ができる
* レスポンシブ対応している
